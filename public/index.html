CyberSense

CyberSense is an interactive and gamified platform
aimed towards spreading cybersecurity awareness in a user-friendly manner.

Here we have 4 different games with 8 questions each on the specified topic.
The quizzes have been designed to help the user understand common cyber security measures and adapt them in their day-to-day working.
After a game ends we display more information on the topic, if the users want to know more.

The intent behind Cyber Sense is to educate the users about the Cyber etiquettes thereby making them more sensible.

Enjoy the gamified experience!
